# Introductron
Aplicación de prueba del framework Electron

### Instalación

```
# Node.js
wget -qO- https://deb.nodesource.com/setup_4.x | sudo bash -
sudo apt-get install -y nodejs

# Aplicación
npm install
```

### Ejecución

```
npm start
```

### Construir

```
npm run build
```
